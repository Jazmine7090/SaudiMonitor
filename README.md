## 📋 What is Saudi Monitor?

Saudi Monitor is an **AI-powered, real-time intelligence dashboard** specifically tailored for the **Saudi Arabia and Gulf region**. It aggregates **450+ curated news feeds** across 11 specialized categories, synthesizes intelligence with AI, and visualizes geopolitical data on interactive 3D/2D maps.

### Key Features

- **🗺️ Dual Map Engine** — 3D globe (globe.gl) and WebGL flat map (deck.gl) with 45+ data layers
- **📰 450+ Saudi/GCC Feeds** — Arab News, Al Arabiya, Reuters MENA, Asharq Business, Saudi Press Agency, and 445+ more
- **🔗 Cross-Stream Correlation** — Military, economic, energy, and security signal convergence
- **📊 Vision 2030 Tracking** — NEOM, The Line, PIF investments, Aramco, Tadawul stocks
- **⚡ Real-Time Data Layers**:
  - Yemen & Houthi Red Sea attacks
  - Iran-Saudi relations & Gulf naval activity
  - Energy (OPEC+, Aramco, oil prices)
  - Markets (Tadawul, GCC exchanges, Bitcoin)
  - Hajj & pilgrimage logistics
  - Saudi sports mega-events
- **🤖 AI-Powered Insights** — Ollama/Qwen local LLM for intelligence assessments (no API keys required)
- **🌐 Arabic Language Support** — Native Arabic feeds with RTL support
- **📱 Multi-Device** — Web app, PWA, and native desktop app (Tauri 2)

**Live Demo:** https://saudi-monitor-gla3haftp-jazmine7090s-projects.vercel.app/
